overridevirtual

Creates the byte buffer containing the command to write from an Adesto RM25C515C chip. The write frame has following format <0x02, second address byte, first address byte, value to write>

Parameters
addressthe address is subdivided into the first and second byte and copied into the sendBuffer.
valueToWritethe value which should be written to the address.
returnSendBufferthe buffer containing the commands to send over the SPI interface.
sendBufferSizethe size of the command buffer. To indicate how many bytes should be sent.
Returns
MEM_NO_ERROR if no error occurred otherwise return error code.
